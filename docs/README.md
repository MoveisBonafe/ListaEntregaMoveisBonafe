# Móveis Bonafé - Gerador de Lista de Entrega

Este é um aplicativo para gerar listas de entrega a partir de arquivos Excel para a empresa Móveis Bonafé.

## Arquivos disponíveis

- **index.html** - Página inicial do aplicativo
- **app.html** - Interface principal do aplicativo
- **modelo.html** - Modelo de documento para visualização
- **versao_final.html** - Versão mais robusta e completa do aplicativo

## Versão Final

A versão_final.html inclui:

- Interface moderna e intuitiva
- Suporte para diversos formatos de planilha
- Detecção automática de cabeçalhos
- Prévia do documento antes de baixar
- Geração de documento Word (.doc) com formatação correta

## Como Usar

1. Acesse o aplicativo pela página inicial
2. Carregue o arquivo Excel contendo as abas "LISTA POR PEDIDO" e "CARGA"
3. Visualize a prévia do documento
4. Clique em "Gerar Documento" e depois "Baixar Documento"

## Requisitos

O arquivo Excel deve conter:
- Uma aba chamada "LISTA POR PEDIDO" com os dados dos clientes
- Uma aba chamada "CARGA" com os dados para a seção TOTAL
- Colunas para CE, MG, TB e IM
